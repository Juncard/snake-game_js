Snake Game

Instrucctions

Use the arrows to move the snake

If you touch the borders or the body of snake, you lose

This game has been done in javascript. You need a server to run. To use NodeJS you need to install the dependency:

npm install -g http-server

And for run open a command line and in the project folder type:

http-server -p 8000

Now go to localhost:8000 and you can play

Play with the Snake!!!
